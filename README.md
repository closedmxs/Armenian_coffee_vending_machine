# Armenian Coffee Vending Machine Simulator
This is the simulator of the Armenian coffee vending machine. Discover what could be the most important thing people have ever created — coffee vending machine, the foundation of daily life and culture.
While running the simulator, typical Armenian music plays in the background to create a more realistic gameplay experience.

The app consists of several menus. When the program starts, the main menu is displayed, showing the coffee machine. Users can explore various sections and functionalities by navigating through the menu options.
